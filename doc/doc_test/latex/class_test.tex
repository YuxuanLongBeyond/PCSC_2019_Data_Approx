\doxysubsection{Detailed Description}
This class is used for approximating test data and analysis of the result of data approximation The main functions of this class are\+:
\begin{DoxyEnumerate}
\item Generate approximated data by a given type of approximation method and relevant parameters
\item Calculate the error between the actual values and the approximated values 
\end{DoxyEnumerate}
