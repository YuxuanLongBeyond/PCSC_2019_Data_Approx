Detailed Description

In this class, we aim to solve data fitting problem. Given a set of 2D points {(x_i, y_i)}, we want to approximate a function f s.t. f(x_i) = y_i, where f can have its parameter w.

This class provides the implementation of four methods for data fitting (or interpolation):

  1. Polynomial fitting, i.e. polyfit, polyval
  2. Linear piecewise interpolation, i.e. interp1
  3. Cubic spline interpolation, i.e. spline
  4. Fourier interpolation, i.e. dct_fit, dct_val

Note that the usages of the previous three methods are very similar to the ones in Matlab.
