\doxysubsection{Detailed Description}
This class builds mainly for matrix operations, which contain\+:
\begin{DoxyEnumerate}
\item Construction of matrix based on 2D array in C++.
\item Basic matrix operations, like assignment, addition, subtraction and multiplication.
\item Auxiliary matrix operations, like transpose, diagonal element addition and row exchange.
\item Advanced matrix operations, like Gaussian elimination for solving linear system and least squares. 
\end{DoxyEnumerate}
